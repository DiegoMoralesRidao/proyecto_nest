## Descripción

Este es un proyecto de NestJS que proporciona una API para gestionar una colección de **Películas** y **Pokemon**.

- **Películas**: Permite crear, listar (con filtros por título y fecha), actualizar y eliminar películas.
- **Pokemon**: Permite gestionar datos de Pokemon, incluyendo nombres, tipos y HP, con capacidades de filtrado.


## Configuración del Proyecto

```bash
$ npm install
```

## Compilar y ejecutar el proyecto

```bash
# desarrollo
$ npm run start

# modo watch (observación)
$ npm run start:dev

# modo producción
$ npm run start:prod
```

## Ejecutar pruebas

```bash
# pruebas unitarias
$ npm run test

# pruebas e2e (extremo a extremo)
$ npm run test:e2e

# cobertura de pruebas
$ npm run test:cov
```

Si buscas una plataforma en la nube para desplegar tu aplicación NestJS, echa un vistazo a [Mau](https://mau.nestjs.com), nuestra plataforma oficial para desplegar aplicaciones NestJS en AWS. Mau hace que el despliegue sea sencillo y rápido, requiriendo solo unos pocos pasos básicos:

```bash
$ npm install -g @nestjs/mau
$ mau deploy
```

Con Mau, puedes desplegar tu aplicación con solo unos clics, permitiéndote centrarte en construir funcionalidades en lugar de gestionar la infraestructura.
